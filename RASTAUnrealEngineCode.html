RASTA Unreal Engine Code

Proof of Concept per spettacoli teatrali in realtà aumentata integrati con EditOR8 e DirectOR8

Nome del Software

RASTAUnrealEngineCode

Descrizione

RASTAUnrealEngineCode è una proof of concept che illustra le potenzialità degli strumenti EditOR8 e DirectOR8 per la creazione di spettacoli teatrali in realtà aumentata.

Include una serie di set virtuali progettati per consentire al performer di interagire direttamente con gli elementi della scena.

Il software è pensato per essere utilizzato in combinazione con DirectOR8 per esperienze teatrali immersive e integrate.

Tecnologie Utilizzate

Unreal Engine Blueprints, MetaHumanSDK, TCP Sockets Plugin.
