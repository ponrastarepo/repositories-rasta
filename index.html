Annotatore Semantico

L’annotatore semantico è composto da tre applicazioni web, accessibili tramite il seguente URL: http://rasta.orangee.it:81

Protezione: L’accesso a questo indirizzo è protetto da password.

Funzionalità principali:

  • Gestire gli utenti autorizzati all’uso dell’annotatore.
  • Amministrare le sessioni e il caricamento dei file.
  • Accedere alle due applicazioni secondarie responsabili dell’annotazione vera e propria.

Accesso alle risorse esterne:

Nella pagina "Risorse esterne" sono presenti due pulsanti per avviare rispettivamente Simple Annotation Server e Brat. Anche queste due applicazioni effettuano una verifica dell’utenza: se l’utente non risulta autorizzato, verrà automaticamente reindirizzato alla pagina di login.

Esecuzione locale:

Essendo un sistema basato su applicazioni web, non è possibile eseguirlo localmente in modo diretto.

Nonostante tutte le componenti siano distribuite in container Docker, le configurazioni dei percorsi (endpoint) non sono dinamiche. Di conseguenza, per eseguire il sistema in locale è necessario:

  • Ricompilare le applicazioni con i puntamenti adeguati all’ambiente locale.
  • Riconfigurare i web server affinché siano compatibili con l’esecuzione su localhost.
RASTA Content Recommender System

Descrizione: Un sistema di raccomandazione di contenuti multimediali in un percorso statico di visita.

Tecnologie: Python, OWL, REST API

Requisiti Software: Python, requests, flask, flask_cors, owlready2, pykml, networkx

Installazione:

pip install requests flask flask_cors owlready2 pykml networkx
python3 main.py

Contatti: dario.branco@unicampania.it

RASTA App

Descrizione: Un'applicazione scritta in Flutter per la visualizzazione di itinerari turistici.

Tecnologie: Flutter, Dart

Requisiti Software: Flutter, Dart

Installazione: Apk lanciabile già presente nella repository.
