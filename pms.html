Nome del Software

RASTA POI Management System

Descrizione

Il POI Management System (PMS) è progettato per gestire i Punti di Interesse (POI), permettendo ai contributori di creare, modificare e pubblicare informazioni. Il sistema include funzionalità sia di front-end (accessibile al pubblico) che di back-office (per la gestione e l’inserimento dei dati).

Tecnologie Utilizzate

Docker, Azure, Apache Web Server, EMQX, InfluxDB, Telegraf, Grafana, Spring Boot, Spring AI, OpenAI, Apache Mahout, Apache Stanbol (deprecato). Per i dettagli completi si rimanda al Deliverable D1.2.

Requisiti

  • Windows, Mac OS X, o sistemi Unix-based
  • OpenJDK 21 (o Eclipse Adoptium)
  • Apache Maven
  • MySQL (presente in MAMP per installazioni locali)

Installazione

1. Creare il database 'rastapms' con:
   username = rastapms
   password = rastapms

2. Eseguire lo script SQL: rasta-pms.sql

3. Compilare il progetto:
   mvn clean compile

4. Avviare il server:
   mvn spring-boot:run

5. Aprire il browser all’indirizzo:
   http://localhost:8080/rastapms/

Utilizzo

Repository di codice sorgente e installazione:

  • official-rasta-pms
  • rasta-me (servizi REST)
  • official-rasta-or6 (istanza OR6)

L’accesso avviene tramite l’icona con freccia in alto a destra nella homepage dell’applicazione.

Il sistema supporta l’arricchimento semantico con OpenAI e la categorizzazione automatica dei contenuti.

Sono già state inserite 21 descrizioni di POI relativi al Parco Nazionale d’Abruzzo.

Configurazione

L’architettura è completamente containerizzata tramite Docker. I dettagli del deployment sono documentati nel Deliverable D2.6.
