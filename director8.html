DirectOR8

Software per la messa in scena teatrale con attori virtuali e reali integrati

Nome del Software

DirectOR8

Descrizione

DirectOR8 è un software progettato per mettere in scena uno spettacolo teatrale. Utilizza un file di configurazione, creato da EditOR8 a partire da un copione annotato, per definire i comandi e le azioni delle varie scene.

Fornisce al performer tutti gli strumenti necessari per guidare la narrazione in base al file di configurazione che definisce la struttura dell’intero spettacolo.

In ciascuna scena possono avvenire azioni predefinite attivate dai comandi presenti nel file di configurazione.

Le scene possono includere anche un attore virtuale con cui l’attore reale può interagire durante la performance.

Tecnologie Utilizzate

Python e librerie: opencv, mediapipe, pynput, openai, ffmpeg.

Requisiti

  • Sistema validato su Windows 11
  • Installazione di Ffmpeg e aggiunta del percorso alla variabile d'ambiente PATH
  • Caratteristiche minime consigliate:
    • Processore: Intel i7 di 14° generazione
    • RAM: 32 GB
    • Scheda grafica: NVIDIA RTX 4070
    • Disco: HD 2 TB
