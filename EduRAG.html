EduRAG

Interfaccia web e bot Telegram per interrogare sistemi RAG su Vectorize.io

Nome del Software

EduRAG

Descrizione

EduRAG è un repository che contiene sia un’interfaccia web sia un bot Telegram per interrogare un sistema Retrieval-Augmented Generation (RAG) realizzato su Vectorize.io.

Entrambi i software permettono l’interrogazione di un sistema RAG popolato con una collezione documentale dedicata alla storia medievale dei luoghi analizzati nel caso di studio di OR8, con particolare attenzione al Castello di Carlo V di Lecce, sede dell’ultimo incontro con la classe dell’istituto selezionato.

Il bot Telegram integra un modello LM per la conversione Text-to-Speech (TTS), consentendo la lettura vocale delle risposte generate dal sistema RAG.

Tecnologie Utilizzate

NextJS, TailwindCSS, OpenAI gpt-3.5-turbo, vectorize.io, Elevenlabs.
