RASTA

Piattaforma digitale per la generazione di itinerari culturali in Umbria e la gestione di POI e TOI.

Repository GitHub

Informazioni sul Software

Nome del Software: CHIP

Descrizione: Piattaforma digitale per:

  • Generazione di itinerari culturali in Umbria.
  • Gestione di POI e TOI da parte degli amministratori territoriali.

Tecnologie utilizzate:

  • Frontend: Angular 11.2.4 (rastaApp)
  • Backend:
    • Node.js 14.14.5 (POI-TOI-Manager, User-Profile Matchmaker)
    • Python 3.9 (Scorer)
    • Python 3.12 (Routeplanner)

Installazione

rastaApp (Frontend):

  1. Scarica la cartella "rastaApp" dal repository: https://github.com/GDV-UniPG/CHIP
  2. Apri la cartella "rastaApp" da linea di comando.
  3. Esegui:
    • npm install (installa le librerie)
    • ng serve (avvia l'applicazione)

POI-TOI-Manager (Backend):

  1. Scarica la cartella "backend/poi-toi-manager" dal repository.
  2. Apri la cartella "poi-toi-manager" da linea di comando.
  3. Esegui:
    • npm install (installa le librerie)
    • node index (avvia il server)

User-Profile Matchmaker (Backend):

  1. Scarica la cartella "backend/user-profile-matchmaker" dal repository.
  2. Apri la cartella "user-profile-matchmaker" da linea di comando.
  3. Esegui:
    • npm install (installa le librerie)
    • node index (avvia il server)

Scorer (Backend):

  1. Scarica la cartella "backend/scorer" dal repository.
  2. Apri la cartella "scorer" da linea di comando.
  3. Crea un ambiente virtuale (consigliato):
    • python3.9 -m venv venv
  4. Attiva l'ambiente virtuale:
    • Linux: source venv/bin/activate
    • Windows: venv\Scripts\activate
  5. Esegui:
    • pip install --upgrade pip setuptools wheel
    • pip install -r requirements.txt
    • pip install gensim
    • python -m nltk.downloader punkt
    • python server.py (avvia il server)

Routeplanner (Backend):

  1. Scarica la cartella "routeplanner" dal repository.
  2. Crea un ambiente virtuale (consigliato):
    • python3.12 -m venv venv
  3. Attiva l'ambiente virtuale:
    • Linux: source venv/bin/activate
    • Windows: venv\Scripts\activate
  4. Esegui:
    • pip install --upgrade pip setuptools wheel
    • pip install -r requirements.txt
    • python server_vrp.py (avvia il server)

Uso

POI-TOI-Manager: Dopo l'avvio locale, accedi alla documentazione API tramite: http://localhost:3000/

User-Profile Matchmaker: Dopo l'avvio locale, accedi alla documentazione API tramite: http://localhost:3200

Routeplanner: Contatta il server tramite API.

Configurazione

rastaApp: Modifica "rastaApp/src/assets/environments/environment.prod.ts" con gli indirizzi dei moduli backend:

  • TRAVEL_PLANNER_ADDRESS
  • POI_TOI_MANAGER_ADDRESS
  • POI_TOI_MATCHER_ADDRESS

POI-TOI-Manager: Crea un file ".env" in "backend/poi-toi-manager" con:

  • PORT_REST, SECRET_JWT, DB_PASSWORD, ecc.

User-Profile Matchmaker: Crea un file ".env" in "backend/user-profile-matchmaker" con:

  • PORT, GRPC_SERVER_PORT, GRPC_SERVER_HOST

Routeplanner: Inserisci nel codice le costanti e carica i file necessari.
