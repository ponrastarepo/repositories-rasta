RASTA

Avatar 3D per Lingua dei Segni Italiana (LIS)

Descrizione

Lo strumento presenta un'applicazione web che implementa un Avatar 3D in grado di riprodurre i gesti necessari per rappresentare i segni della Lingua Italiana dei Segni (LIS).

Tecnologie utilizzate

  • Riferimento segni LIS: Spread the Sign (European Sign Language Center)
  • Creazione segni: HamNoSys e HamNoSys Palette
  • Rappresentazione segni: SigML
  • Traduzione HamNoSys a SigML: HamNoSys2SigML
  • Applicazione: HTML, CSS, JavaScript
  • Avatar 3D e animazione: Libreria JavaScript CWASA

Requisiti

  • Python installato (nessuna libreria specifica richiesta)

Per creazione Segni

  • HamNoSys Palette o font HamNoSys (solo MacOS)
  • HamNoSys2SigML (conversione HamNoSys a SigML)

Per utilizzo Avatar

  • Browser web

Installazione

  • Scarica il repository

Uso

  1. Apri la directory principale del repository: cd Avatar\ LIS
  2. Avvia il server: python -m http.server 8000
  3. Apri il browser all'indirizzo: http://localhost:8000
  4. L'avatar animerà il file demo/demo.sigml
