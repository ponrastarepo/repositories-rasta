EditOR8

Componente software per la gestione di copioni annotati e configurazione di DirectOR8

Nome del Software

EditOR8

Descrizione

EditOR8 è un componente software che, operando in collaborazione con DirectOR8, permette di mettere in scena uno spettacolo teatrale. Consente di creare una struttura dati a partire da un copione annotato in formato PDF, che può essere utilizzata per configurare DirectOR8.

Tecnologie Utilizzate

Python, PdfReader (libreria Python).

Requisiti

  • Python
